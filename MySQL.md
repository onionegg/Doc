# MySQL

1. database

## keys
> `primary key`獨特性資料、資料不能重複、不能是NULL值、通常把id欄位當作`primary key`
>
> `foreign key`table之間串聯使用
## data type
### `INT`數字型態
> `INT`存數字型態
>
> `BIGINT`存大數據數字型態
>
> `DECIMAL`存浮點數
>
### `DATETIME`存日期、時間
* DATETIME
    * YYYY-MM-DD HH:MM:SS
* DATE
    * YYYY-MM-DD
* TIME
    * HH:MM:SS
### `VARCHAR(M)`純字串
